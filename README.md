# 基于人脸识别的实验室智能门禁系统 python577

## 项目概述

本项目是基于人脸识别技术的实验室智能门禁系统，旨在提升实验室安全性与管理效率。

## 技术栈

- **后端语言**：Python
- **数据库**：MySQL
- **框架**：Django
- **架构类型**：B/S（浏览器/服务器）

## 功能模块

- **登录模块**：用户登录系统
- **首页模块**：展示系统基本信息
- **预订实验室模块**：在线预约实验室
- **预约记录模块**：查看和管理预约记录
- **考勤签到模块**：通过人脸识别实现考勤签到

## 系统角色

- **普通用户**：可登录系统，预订实验室，查看预约记录，进行考勤签到
- **管理员**：除拥有普通用户权限外，还可以管理实验室预订信息及用户考勤数据

## 运行环境

- **操作系统**：兼容主流操作系统
- **Python 版本**：3.x
- **依赖管理**：通过 pip 管理

## 部署步骤

1. 安装 Python 环境（3.x版本）
2. 安装 MySQL 数据库
3. 使用 pip 安装依赖包
4. 初始化数据库及表结构
5. 运行 Django 项目
